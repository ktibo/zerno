# zerno
Обработка обмена данными с ФГИС Зерно через 1С.
Написана для нетиповой самописной конфигурации.

Возможности:
* Работа с СДИЗ (создание, погашение, чтение)
* Работа с партиями (аналогично)
* Получение справочников
* Подписание запросов с помощью сертификата ЭЦП
* Чтение исходных текстов запросов в формате XML
* Работа через фоновое задание
